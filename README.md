# Frontend AI Capstone

This repository contains my capstone project for the Frontend AI Engineering track. The project will showcase AI-assisted frontend development practices, modern web technologies, and Git-based collaboration throughout the program.

## Objectives

- Learn AI-assisted software development workflows.
- Build a modern frontend application using industry best practices.
- Maintain a clean and well-documented Git history using Conventional Commits.
- Develop a production-ready capstone project.

## Planned Tech Stack

- React
- TypeScript
- Vite
- Tailwind CSS
- Git & GitHub
- Cursor AI

## Project Status

🚧 Initial project setup is in progress.

## Installation

```bash
git clone https://github.com/binibiningjenna/frontend-ai.git
cd frontend-ai
npm install
npm run dev
```

## Git Workflow

This project follows the [Conventional Commits](https://www.conventionalcommits.org/) specification for maintaining a clear and consistent commit history.
